# 📊 Employee Attrition Analysis

## 📌 Overview

This project analyzes employee attrition using the IBM HR Analytics dataset. The goal is to identify the factors that contribute to employee turnover through data cleaning, exploratory data analysis (EDA), and data visualization using Python.

---

## 🎯 Objectives

- Analyze employee attrition trends.
- Identify key factors influencing employee turnover.
- Perform exploratory data analysis (EDA).
- Generate meaningful business insights using visualizations.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset

**Dataset:** IBM HR Analytics Employee Attrition Dataset

The dataset contains employee information such as:

- Age
- Department
- Job Role
- Monthly Income
- Job Satisfaction
- Overtime
- Years at Company
- Attrition Status
- Education
- Performance Rating

- Job Satisfaction Analysis
- Employee Age Distribution
- Correlation Analysis

---

## 📈 Key Findings

- Overall employee attrition rate is **16.12%**.
- Employees who work overtime have a higher attrition rate.
- The Sales and Research & Development departments experience the highest number of employee resignations.
- Employees with lower monthly income tend to leave more frequently.
- Job satisfaction plays an important role in employee retention.
